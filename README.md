Random string generator make from colours, animals and numbers.

All code is PSR-2 friendly

## Install

Via Composer

``` bash
$ composer require jamosaur/randstring
```

## Usage

``` php
$rand = new Jamosaur/Randstring/Randstring;
echo $rand->generate();
```
